== Description ==

CF Block is a custom Gutenberg Block That has the following  upgradation to be followed they are 
1) User Can Select the Contact Form Layout One-Column & Two-Column.
2) User can select the Contact Form Fields.
3) User Can Add Custom Sender Email ID Form.
4) User Can Add the Email Subject From This Gutenberg Block.
5) User Can Enable & Disable Captcha Settings From the Gutenberg Block.
6) User Can Add the Recaptcha to the Form.
7) User can add the Border Color,Border radious,Border Style,Block Padding,Title color, Sub-title color, Sub-Content Color,Block Background Color.
8) User Can Send the Contact Form Email To Admin From this Gutenberg Block.

FEATURES
Custom Email Receiver Email ID & Email Subject
One-Column & Two-Column Contact Form Layout
Recaptcha Functionality
Background color
Title text color

== Installation ==

TO INSTALL THE PLUGIN AUTOMATICALLY:
Through WordPress admin, use the menu: Plugin > Add New
Search for CF Block
Click on install then click the activation link
